my_package.trapezoid.trapzf(f, a, b, npts=100)¶

Integrate a function using the trapezoid rule.

Parameters:

f : callable f(x)

Function to integrate.

a, b : float

Interval over which to integrate f.

npts : int, optional

Number of equally spaced points at which to sample f at (between a and b).
Returns:

float

Integral of f over the interval \([a, b)\).

Notes

Just to show that it can be done, a LaTeX equation:

\[\int_\mathrm{a}^\mathrm{b} f(t) dt\]
